Input, Output, and Type Conversion

input()

Reads a value typed by the user as a string.

name = input("Enter your name: ")
print("Hello, " + name)  # Output: Hello, Alex

print()

Displays text or variables on screen.

points = 10
print("You scored", points, "points")  # Output: You scored 10 points

int(), float(), str(), bool()

Convert between data types.

age = int("18")
height = float("1.75")
text = str(123)
is_ready = bool(1)
print(age, height, text, is_ready)  # Output: 18 1.75 123 True

Selection

if, elif, else

Run different code depending on a condition.

score = 72
if score >= 80:
    print("Distinction")
elif score >= 60:
    print("Merit")
else:
    print("Pass")  # Output: Merit

match case

Matches one value against fixed cases (Python 3.10+).

grade = "B"
match grade:
    case "A":
        print("Excellent")
    case "B":
        print("Good")
    case _:
        print("Needs improvement")  # Output: Good

Iteration

for

Repeats actions a known number of times.

for i in range(3):
    print("Loop", i)  
# Output:
# Loop 0
# Loop 1
# Loop 2

while

Repeats actions while a condition is true.

count = 0
while count < 3:
    print("Counting:", count)
    count += 1
# Output:
# Counting: 0
# Counting: 1
# Counting: 2

Functions

def

Defines reusable blocks of code.

def add(a, b):
    return a + b

print(add(3, 5))  # Output: 8

Standard Libraries

import math

Use mathematical functions and constants.

import math
print(math.sqrt(16))   # Output: 4.0
print(math.pi)         # Output: 3.141592653589793

import random

Generate random numbers.

import random
print(random.randint(1, 6))  # Output: random int between 1 and 6
print(random.random())       # Output: random float between 0 and 1

Built-in Functions

range()

Creates a sequence of numbers, often used in loops.

for n in range(1, 4):
    print(n)
# Output: 1 2 3

len(), max(), min(), round()

Return length, largest, smallest, and rounded values.

nums = [4, 7, 2]
print(len(nums))   # Output: 3
print(max(nums))   # Output: 7
print(min(nums))   # Output: 2
print(round(3.1416, 2))  # Output: 3.14

chr(), ord()

Convert between characters and ASCII codes.

print(chr(65))  # Output: A
print(ord('A')) # Output: 65

String Handling

upper(), lower(), isalpha(), isdigit()

Change case or check content of strings.

word = "Hello"
print(word.upper())   # Output: HELLO
print(word.isalpha()) # Output: True
print("123".isdigit())# Output: True

find(), replace(), split(), strip(), format()

Locate, modify and tidy strings.

text = "apple, banana, cherry"
print(text.find("banana"))     # Output: 7
print(text.replace("banana","orange"))  # Output: apple, orange, cherry
print(text.split(", "))        # Output: ['apple','banana','cherry']
print("  spaced  ".strip())    # Output: spaced
print("Hello {}".format("World"))  # Output: Hello World

Data Structures

Lists

Store ordered collections of items.

fruits = ["apple", "banana"]
fruits.append("cherry")
print(fruits[1])  # Output: banana
print(len(fruits)) # Output: 3

Dictionaries

Store key-value pairs.

person = {"name": "Alex", "age": 20}
print(person["name"])  # Output: Alex

Text Files

Open, write, read and close files safely.

# Write to a file
file = open("example.txt", "w")
file.write("Hello File!")
file.close()

# Read from the file
file = open("example.txt", "r")
print(file.read())  # Output: Hello File!
file.close()

Dates and Times

Work with current date and time.

from datetime import datetime
now = datetime.now()
print(now.strftime("%Y-%m-%d"))  # Output: 2025-10-05
# Convert string back to datetime
dt = datetime.strptime("2025-10-05", "%Y-%m-%d")
print(dt.year)  # Output: 2025
